# Clawd Configuration

Versioned configuration and documentation for Clawd AI assistant.

## ⚠️ Security Notice

This repository contains **NO credentials, tokens, or API keys**. All sensitive data is stored externally in `~/.config/` and `~/.clawdbot/credentials/` directories.

## Structure

- `docs/` - Documentation and guides
- `skills/` - Skill definitions and scripts (credential-free)
- `scripts/` - Utility scripts (reference external credentials)
- `memory/` - Memory file templates and examples

## Setup

1. Clone this repo
2. Install credentials separately (not in this repo)
3. Run `clawdbot` commands as documented
